Animates your name. When you move your mouse over your name, bubbles will scatter away and then reassemble.

== Description ==

*   You can add your text for typing animation.
*   You can set height and wight of bubble canvas size.

= Read about Shortcode: =
[typingtext text='Text Here' type_speed='200' type_delay='200' remove_speed='50' remove_delay='500' cursor_speed='200' loop='true']

*   Text (optional) : set your text you want to animate on it other vice it takes "Typing Text Animation" by default.
*   type_speed (optional) : set typing speed in millisecond. Default values is 200.
*   type_delay (optional) : set typing delay time in millisecond. Default values is 200.
*   remove_speed (optional) : set text remove speed time in millisecond. Default values is 50.
*   remove_delay (optional) : set text remove delay time in millisecond. Default values is 500.
*   cursor_speed (optional) : set cursor blink speed time in millisecond. Default values is 200.
*   loop (optional) : set loop animation true or false. Default values is true.


== Installation ==

1. Upload the entire `typing-text` folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy! ;)
